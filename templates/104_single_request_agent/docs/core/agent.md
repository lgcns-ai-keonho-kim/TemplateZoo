# Core Agent

## 그래프 목적

사용자 요청 1건에 대해 다음 순서로 응답을 만든다.

1. `intent_classify`: 입력을 요약, 번역, 형식 글쓰기, 일반 응답 중 하나로 분류
2. `intent_prepare`: 분류 결과를 표준 토큰과 작업 지시문으로 정규화
3. `response`: 의도별 지시문을 반영해 최종 응답 생성

## 실행 특징

1. 그래프는 여전히 스트림 이벤트를 생성한다.
2. 외부로 SSE를 내보내지 않고 `AgentService`가 내부에서 직접 집계한다.
3. 최종 상태는 정상 완료 시 `COMPLETED`만 반환한다.
