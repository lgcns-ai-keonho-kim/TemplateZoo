## Contribution Guide

> ![IMPORTANT] 
> 신규 예제를 작성 할 때는 항상 [BASE TEMPLATE](./000_base_template/)을 먼저 복사하세요.

```bash
cp -r 000_base_template <TEMPLATE_NUMBER>_<STH_TITLE>
```

- <b>`Base Template`</b>은 모든 구현에 사용되는 기본 구조입니다.
- 최대한 구조를 지켜 작성하시되, <b><I>작성이 완료된 다음 불필요한 부분에 대해서 삭제해주세요.</I></b>
